# 会計のテスト用

青山学院大学 会計学の授業内テスト対策ページです。

## ページ構成

- `index.html`: まとめページ
- `test1.html`: 第1回テスト対策。第2回から第6回、財務会計1から5。
- `test2.html`: 第2回テスト対策。第7回から第10回。
- `test3.html`: 第3回テスト対策。第11回から第14回。

## 現在の作成方針

第1回テスト対策は、講義資料と実際に出た問題を反映済みです。

第2回・第3回テスト対策は、まず授業計画画像から読める内容をもとに仮ページを用意します。講義資料が入手できた回から順番に、以下の内容へ差し替えます。

- 講義資料に沿った要点整理
- 用語カード
- 図解
- 記述式の予想問題
- 実際に出た問題のログ

現時点では、第2回テスト範囲のうち第7回「連結会計と税務会計」の講義資料のみ反映済みです。

## 開き方

`index.html` をブラウザで開いてください。

GitHub Pagesで公開する場合は、このリポジトリのPages設定で `main` ブランチのrootを選ぶと表示できます。
